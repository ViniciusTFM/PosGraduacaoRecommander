Público-alvo:

- Profissionais de nível superior diplomados em cursos da área de tecnologia da Informação, como: Sistemas de Informação, Análise e Desenvolvimento de Sistemas, Ciência da Computação, Engenharia de Computação, Ciência da Informação e outras afins;
- Profissionais de nível superior que atuam no mercado como gerentes, coordenadores ou consultores na área de tecnologia da informação, analistas de negócios, analistas de desenvolvimento de sistemas, analistas de suporte técnico e desenvolvedores web.
- Profissionais da área de engenharia de software que já atuam em projetos de TI envolvidos em qualquer etapa do processo de desenvolvimento de software e queiram ampliar e aperfeiçoar seus conhecimentos técnicos sobre   desenvolvimento de aplicações para dispositivos móveis nos mais diversos segmentos. 

Objetivo(s):

- Mostrar a importância do desenvolvimento de aplicativos para dispositivos móveis no contexto corporativo e na capacidade de identificar problemas, oportunidades e propor soluções inovadoras para o mercado; 
- Apresentar conceitos sobre o Processo de Desenvolvimento de software para dispositivos móveis;
- Fornecer uma ampla visão sobre os conceitos relacionados ao desenvolvimento de aplicações móveis;
- Proporcionar competências e conhecimentos aos participantes em técnicas, ferramentas e tecnologias para participar de atividades de concepção, projeto, desenvolvimento, manutenção, gerenciamento, administração e utilização de aplicações  para dispositivos móveis;
- Capacitar o participante para propor soluções para integrar softwares distribuídos do ponto de vista de banco de dados e ambiente web;
- Capacitar o participante para redefinir arquiteturas de softwares legados ou integrá-los à aplicativos móveis;
- Capacitar o participante para atuar forma empreendedora na criação de novos negócios, utilizando os conceitos e as técnicas aprendidos no curso;
- Identificar e discutir conceitos emergentes relacionados à tecnologias para desenvolvimento de aplicações móveis  e verificar seus impactos em novos projetos.

Programa:

- Arquitetura de Aplicações para Dispositivos Móveis 
- Arquitetura e Desenvolvimento de APIs para Back-end
- Banco de Dados para Aplicações Móveis 
- Briefing: identificação das estratégias do negócio e dos requisitos da aplicação
- Computação em Nuvens para Aplicações Móveis
- Desenvolvimento de Aplicações para Android 
- Desenvolvimento de Aplicações para IOS
- Desenvolvimento de Aplicações para Windows Phone
- Desenvolvimento de Aplicações Web para Dispositivos Móveis
- Design de Interação para Aplicações Móveis
- Garantia de Qualidade de Aplicações Móveis
- Integração de Aplicações Móveis com Redes Sociais
- Internet das Coisas: arquitetura, tecnologias e aplicações 
- Metodologia da Pesquisa Científica (EAD)*
- Modelos de Receita e Métricas para Análise de Desempenho
- Plataformas de Desenvolvimento de Aplicações Móveis 
- Projeto Integrado: Mercado, análise, projeto e implementação de uma aplicação
- Segurança em Aplicações e Serviços Móveis
