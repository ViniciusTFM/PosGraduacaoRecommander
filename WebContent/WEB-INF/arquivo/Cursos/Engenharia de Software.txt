Público-alvo:

Profissionais de nível superior com formação básica em Ciência da Computação, Sistemas de Informação, Ciência da Computação e Engenharia e/ou profissionais de outras áreas que atuem no mercado como gerentes ou coordenadores de informática, analistas de negócios, analistas de desenvolvimento de sistemas, analistas de suporte técnico e consultores de sistemas.
Objetivo(s):

O curso tem o objetivo de desenvolver junto aos alunos habilidades de análise e utilização das tecnologias e metodologias da Engenharia de Software com o intuito de aplicá-las nos mais variados contextos organizacionais.
Programa:

- Abordagens Ágeis para Desenvolvimento e Gerenciamento de Software 
- Arquitetura de Aplicações para Dispositivos Móveis
- Arquitetura de Aplicações Web
- BPM- Business Process Management
- Engenharia de Requisitos 
- Gerência de Configuração de Software 
- Implementação e Reutilização de Software
- Interação Homem-Computador
- Metodologia da Pesquisa Científica (EAD)*
- Métricas e Estimativas em Projetos 
- Modelos de Maturidade de Processos de Software 
- Princípios de Gerência de Projetos com PMBOK 
- Processos Prescritivos para Desenvolvimento de Software 
- Projeto (Design) de Software
- Projeto Aplicado em Engenharia de Software
- Qualidade de Projeto e Produto de Software 
- Tecnologias de Bancos de Dados
- Testes Sistemáticos de Software
